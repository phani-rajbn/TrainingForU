Tables in HTML Documents

Important points

  • Tables are used to represent tabular content within a HTML Document
  • <table> tag is used to create a table in the doc.
  • Each row within the table is placed under <tr>
  • Each row will have columns and the data of the columns are provided using <td>
  • The <tr> will have multiple <td> and have end </tr>.
  • If your table needs a border or a extended width, U should set the border attribute to 1 and width to % units of the device width
