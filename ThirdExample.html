Lists in HTML Documents

If U wish to present a list of items in a document U could present it using 2 kinds of listings: ordered list and unordered list.
Ordered lists are used to place the listings thro a numbering format using a tag ol
Unordered lists are placed in bulletted format using a Tag called ul
Each item in the Listing is provided under a li tag. li means listItem
ol contains an attribute called type that will specify how your numbering should happen. Possible values are 1, i, I, a, A. Default is 1
