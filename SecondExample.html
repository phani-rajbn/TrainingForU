More HTML Elements

HTML tags are grouped as Basic tags, Formatting Tags, Phrase Tags, Listing Tags, Table Tables, links and Forms.

Basic Tags

The Basic tags are head, body, headings(h1....h6), divs, p and hr. These tags are basic tags that are found in any kind of HTML documents.

Formatting Tags

These tags are used to format the text of the document. This would include bold(b), underlined(u), itallics(i), strikeThrough(strike) and so forth. These formatting tags are used to present the text in a certain formats.

Phrase Formatting Tags

These Tags are applied on Phrases of text within the HTML Document. These phrases could be for a quotation given by famous persons, a block of programming code to be displayed in the document, a pre arranged presentation for the document and so forth....
